# Hanuman-Ji-Turtle-Art-Python-Project
This is a creative Python project built using the Turtle graphics library, where a devotional artistic illustration inspired by Lord Hanuman Ji is drawn using code.
✨ Features
Black background canvas for high contrast design
Custom face artwork using curves and geometry
Detailed facial parts (eyes, nose, lips, chin, eyebrow)
Smooth turtle animation drawing
Motivational devotional caption in Hindi
Colorful styling (red, orange, white)
🧠 Concepts Used
Python Turtle Graphics
Functions for modular drawing
Loops for curve generation
Coordinate positioning system
Pen control (penup, pendown, width, color)
